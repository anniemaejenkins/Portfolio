Who is Annie, anyways?

Since I first picked up a controller to play games, I have always had an interest in technology. When I discovered blogging on the web as a middle schooler, I became obsessed with code and how I could use it to change the way my layout and words would appear on my page. After graduating high school, I wasn't sure of what I would do about college due to expenses and location, so I stayed home and worked while looking for an alternative to study. The opportunity came along when I discovered The Iron Yard, a coding school in the downtown area of Greenville, SC. I quickly found that solving the same problem in different ways was my favorite aspect of coding and honestly, being able to create a working web site from start to finish is pretty awesome. The community of developers seems inclusive while willing to help others and bringing people together for a common goal is something I care about, so having been able to learn this about code has been very rewarding. I honestly can't wait to see what's next!
